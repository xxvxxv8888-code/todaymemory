# 오늘의 기억 (Today's Memory)

**하루를 한 장의 '기억 카드'로 남기는 감성 라이프로그 앱**

사진이 없어도, 영수증이 없어도, 메모 한 줄만 있어도 오늘을 기록할 수 있습니다.
부담 없이 남긴 하루가 쌓여 1년 뒤·3년 뒤·5년 뒤 다시 보며 추억이 됩니다.

가계부도, SNS도 아닙니다. 주인공은 언제나 **'기억'** 이에요.

---

## ✨ 주요 기능

- **간편한 기록** — 사진·메모·날짜·기분·태그·함께한 사람·장소를 원하는 만큼만. 쓰는 즉시 자동 저장
- **라이프맵** — 한 해를 12조각 퍼즐로. 한 달씩 채워 한 장의 사진을 완성하는 성취감
- **캘린더 / 지도 / 회고** — 날짜별, 지역별로 추억을 탐색하고 'N년 전 오늘'을 다시 만나기
- **통계 · 앨범 · 검색** — 가장 많이 간 곳, 자주 쓴 태그, 전체 사진과 빠른 검색
- **영수증 인식(OCR)** — 상호명·날짜·금액 자동 인식. 모든 인식은 **기기 안에서만** 처리
- **백업 & 복원** — 백업 파일 하나로 안전하게 보관하거나 새 기기로 이전
